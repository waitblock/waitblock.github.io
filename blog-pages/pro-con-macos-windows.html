The Pros and Cons of MacOS and Windows

Published January 2nd, 2021.

Last year, I got my first MacBook and decided to make the switch to MacOS instead of installing Windows on the MacBook. I had never actually used any other operating system other than Windows up to that point. I had used Raspbian (a Linux distro for the Raspberry Pi based off of Debian) because I got a Raspberry Pi, but other than that, I hadn't used any other operating system. So let me tell you my pros and cons to each MacOS and Windows.

Let me start by telling you the general details and backstory for each operating system.

Windows

  • Made by Microsoft
  • Runs on the NT kernel
  • Flexible hardware options (i.e. the ability to run on any computer components)

MacOS

  • Made by Apple
  • Runs on a kernel based off of parts of BSD and the XNU kernel called Darwin
  • Closed hardware options (i.e. can only be run on specific computer components)

Windows is probably the most common operating system on personal computers at the time of writing, as it is included on most personal computers and has the ability to be installed on any hardware. In addition, the high support for software on Windows adds to the reasons why Windows is the most popular operating system. MacOS, on the other hand, can only be installed on Apple-specific devices. Even though there are ways to install MacOS onto a non-Apple piece of hardware, usually called a Hackintosh, most of the time these installations are too difficult for the end user.

But of course, there must be a rhyme or reason to why some people prefer MacOS over Windows even though MacOS has hardware limitations. Mac computers are based off of the BSD operating system, which is based off of Linux. Linux has the advantage of being older and therefore being more secure and stable due to years of development. In addition, Linux has a simpler feature-set than Windows, meaning that there are less bugs to fix, thus leading to a more stable and secure operating system. In addition, with a Linux kernel you can poke around with the operating system in any way you want. This may not be too helpful to the common user, but for developers this is very helpful because you can change hidden files and such to test applications that require system privileges, etc. For the end user, this feature could be used to shutdown all of the system telemetry. On Windows, there is a switch to turn on sending data reports to Microsoft. But who knows if that switch actually turns off data reports? So, as you can see, even though it may seem like at first Windows would be the most obvious go-to operating system, MacOS has advantages in some areas where Windows lacks.

Below are the listed pros and cons for each operating system, which I will then summarize.

Windows:

Pros:

  • Flexible hardware options
  • High support for software and hardware
  • Gaming

Cons:

  • High resource use (i.e. uses a lot of processing power)
  • Poor security
  • Poor stability

As mentioned above, Windows is great because you have flexible hardware options and a high support for software. In addition to that, hardware vendors are forced to make their new components compatible with Windows because of the large market share that Windows holds for personal computer operating systems. Another advantage of Windows is the ability for high-resource games to run with good results. Since Windows can run on any hardware, it is commonly used to run high-resource games as the hardware prices for an Apple product with similar hardware specifications to run the same game could be way above the price point for the Windows computer. One problem with Windows is its high-resource usage. This is due to the fact which Microsoft claims is to allow for features that users wanted added to Windows to be used smoothly. However, very few people actually use those features. In addition, since Windows is jam-packed with features and such, there are many bugs which are yet to be fixed. However, this leads to bad security and bad stability compared to MacOS systems. If you are a Windows user, think about how many times you've had to restart your computer. This is because of all of the bugs that Microsoft is yet to fix.

MacOS:

Pros:

  • Good security
  • Good stability
  • Compatibility with other Apple devices/services
  • Comes with useful (and free) productivity applications
  • The ability to "poke" around in the system

Cons:

  • High cost
  • Closed hardware options
  • Unappealing to the gaming industry
  • Limited (and sometimes frustrating) hardware upgrades/repairs

As said in the introduction, MacOS has good security and good stability due to its simpler user interface. In addition, it seamlessly integrates with other Apple devices/services via applications such as Airdrop, iCloud, etc. Also, as said above, developers have the ability to poke around in the system and test out features that might require integration with the system files, etc. However, not everything can be sunshine and daisies. Since MacOS can only run on specific hardware, and making a Hackintosh is very difficult, there is less appeal of Apple products to the gaming market because of the limited hardware options. In addition, as mentioned in the Windows summary, Apple hardware is generally very expensive compared to Windows hardware of similar specification. Since Apple creates the hardware, they're also the one who usually fixes the hardware. However, Apple repairs are often expensive, and sometimes deceiving. They may charge you more than needed to repair a product, and will often just recommend you to buy another computer, which you will again, eventually need to repair or replace.
